14- ls
15- tmux 
16- streamlit run autoscout.py (APP klasörü içerindeyken çalıştırılmalı)
17- CTRL+B, D (tmux detached)

Tmux CheatSheet
1- start new:
-tmux
2- start new with session name:
-tmux new -s myname 
3- attach:
-tmux a  #  (or at, or attach)
4- attach to named:
-tmux a -t myname
5- list sessions:
-tmux ls
6- kill session:
-tmux kill-session -t myname
NOT: (myname) session adı, tmux kill-session -t den sonra sadece session başındaki numarayı yazarakta sonlandırabiliriz.
